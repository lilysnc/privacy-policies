## 1. 기본 원칙

`냥냥타워: 우다다팡`의 운영 주체인 릴리에스앤씨(공개 개발자명: 릴리게임즈, 이하 “운영자”)는 이용자의 개인정보를 중요하게 생각하며, 개인정보 보호법 등 관련 법령을 준수합니다.

현재 버전은 로그인, 회원가입, 광고, 인앱 결제, 분석, 원격 오류 수집, 푸시 알림, 클라우드 저장 또는 이용자 간 상호작용 기능을 제공하지 않습니다. Google Play용 Android 앱은 게임 이용 과정에서 운영자의 서버로 개인정보나 게임 데이터를 전송하지 않습니다.

이 방침은 다음 배포 환경에 적용됩니다.

- Google Play에서 배포하는 Android 앱
- 앱인토스에서 제공하는 게임 미니앱
- 개발 및 품질 검증을 위한 브라우저 버전

배포 플랫폼 사업자가 자신의 서비스 운영을 위해 독립적으로 처리하는 정보에는 각 플랫폼의 개인정보처리방침이 적용됩니다.

앱인토스 버전에서 사용하는 사용자 식별 해시값은 토스의 `getUserKeyForGame` API가 반환하는 값입니다. 이 값은 `냥냥타워: 우다다팡` 미니앱 안에서만 유효한 가명 식별자이며, 운영자는 이를 별도 서버로 전송하거나 다른 서비스의 정보와 결합하지 않습니다.

## 9. 자동 수집 장치와 네트워크

현재 앱은 맞춤형 광고, 이용자 추적, 웹 로그 분석을 위한 쿠키, 픽셀, 광고 ID 또는 분석 SDK를 사용하지 않습니다.

Android 앱에는 Capacitor WebView가 앱에 포함된 로컬 게임 자산을 제공하기 위해 일반 `android.permission.INTERNET` 권한을 선언하지만, 현재 release 버전은 외부 콘텐츠, 원격 코드, 광고 또는 운영자 API 서버에 접속하지 않습니다.

개발용 브라우저 버전은 게임 진행과 음향 설정을 보존하기 위해 브라우저 `localStorage`를 사용할 수 있습니다. 이 저장소는 추적이나 맞춤형 광고에 사용되지 않습니다.

## 10. 보유 기간과 파기 방법

운영자 서버에는 앱의 개인정보나 게임 데이터가 저장되지 않습니다.

이용자는 다음 방법으로 로컬 게임 데이터를 삭제할 수 있습니다.

- Google Play Android 앱: Android 설정에서 `냥냥타워: 우다다팡`의 저장공간·앱 데이터를 삭제하거나 앱을 제거
- 앱인토스 버전: 토스 앱의 해당 저장소가 초기화되거나 토스 앱이 제거되면 SDK `Storage` 데이터 삭제
- 브라우저 개발 버전: 해당 사이트의 저장 데이터 또는 `localStorage` 삭제

삭제한 진행 데이터는 운영자가 보유한 서버 사본이 없으므로 복구할 수 없습니다.

문의 과정에서 받은 전자적 정보는 보유 기간이 끝나면 복구하기 어려운 방법으로 삭제합니다.
