## 1. 기본 원칙과 적용 범위

`각본대로`의 운영 주체인 릴리에스앤씨(공개 개발자명: 릴리게임즈, 이하 “운영자”)는 이용자의 개인정보를 중요하게 생각하며 개인정보 보호법 등 관련 법령을 준수합니다.

이 방침은 다음 배포 환경에 적용됩니다.

- Google Play에서 배포하며 게임 웹 자산을 APK/AAB에 포함한 Android WebView 앱
- 앱인토스에서 제공하는 게임 미니앱
- 개발 및 품질 검증을 위한 브라우저 실행본

Google Play Android 앱은 게임 화면, 대사와 이미지 등 모든 웹 자산을 앱 패키지 안에서 읽습니다. 앱은 `android.permission.INTERNET`을 요청하지 않으며 게임 데이터를 외부 서버로 전송하지 않습니다. 앱인토스 버전도 게임 코드에서 별도 외부 API, 광고 SDK 또는 분석 SDK를 호출하지 않고 배포 번들에 포함된 자산으로 실행됩니다.

현재 서비스에는 회원가입, 로그인, 프로필, 채팅, 이용자 게시물, 인앱 결제, 클라우드 저장, 운영자 자체 서버, 광고 SDK, 이용 분석 SDK 또는 원격 오류 수집 SDK가 없습니다.

Google Play, 토스, GitHub 또는 이메일 제공자가 자신의 서비스 운영을 위해 독립적으로 처리하는 정보에는 각 제공자의 개인정보처리방침과 약관이 함께 적용됩니다.

## 3. 기기 안에 저장되는 정보

서비스는 게임 기능과 이용자 설정을 유지하기 위해 다음 정보를 이용자의 기기 또는 실행 환경에 저장할 수 있습니다.

| 저장 항목 | 목적 | 저장 위치 | 보유 기간 |
| --- | --- | --- | --- |
| 엔딩 도감 정보 | 이용자가 확인한 배드엔딩·해피엔딩과 도감 달성 상태 복원 | Android WebView, 앱인토스 또는 브라우저의 Local Storage (`gbd_dex_v1`) | 앱 데이터·사이트 데이터 삭제 또는 앱 제거 시까지 |
| 음소거 설정 | 효과음과 배경음의 켜짐·꺼짐 상태 복원 | 같은 Local Storage (`gbd_mute`) | 같은 기간 |
| 현재 플레이 상태 | 선택한 상황·배역·컷, 현재 선택지, 힌트와 이어하기 사용 여부 관리 | 실행 중인 앱 또는 웹 페이지의 메모리 | 앱·페이지 종료 시까지 |
| 정적 게임 자산 | 네트워크 없이 게임 화면, 대사와 이미지를 제공 | Google Play APK/AAB 또는 앱인토스 배포 번들 | 앱·미니앱 업데이트 또는 제거 시까지 |

엔딩 도감에는 게임 내부 엔딩 식별자만 저장되며 이름, 이메일, 전화번호 또는 계정 ID가 포함되지 않습니다. 운영자는 이 로컬 정보를 자체 서버로 전송하거나 이용자의 실제 신원과 결합하지 않습니다.

Google Play Android 앱은 쿠키와 제3자 쿠키를 허용하지 않습니다. 앱인토스와 일반 브라우저에서 플랫폼 또는 브라우저가 자체 기능을 위해 관리하는 저장소와 캐시는 해당 플랫폼·브라우저의 설정과 정책을 따릅니다.

## 4. 운영자가 수집하지 않는 정보

운영자는 서비스 기능을 통해 다음 정보를 직접 입력받거나 운영자 자체 서버에 저장하지 않습니다.

- 이름, 생년월일, 성별과 계정 비밀번호
- 이메일 주소와 전화번호
- 정확한 위치 또는 대략적인 위치
- 주소록과 연락처
- 사진, 동영상과 파일
- 카메라와 마이크 입력
- 결제, 금융과 거래 정보
- 광고 식별자와 기기 식별자
- 건강, 생체정보 등 민감정보
- 이용자의 실제 신원과 연결된 게임 진행 기록

서비스에는 이용자가 이러한 정보를 입력하는 화면이 없습니다. 이용자가 지원 이메일에 자발적으로 정보를 포함하면 제6조에 따라 해당 문의 정보가 처리될 수 있습니다.

## 5. 광고, 분석과 네트워크 사용

현재 서비스는 Google Mobile Ads(AdMob), Google UMP, Firebase Analytics, Firebase Crashlytics 또는 다른 광고·분석·원격 오류 수집 SDK를 포함하지 않습니다. 운영자는 자체 추적용 쿠키, 픽셀, 광고 서버 또는 행동 분석 서버를 사용하지 않습니다.

게임 화면의 `광고 보고 이어하기` 버튼은 실제 광고를 요청하거나 표시하지 않습니다. 이용자가 버튼을 선택하면 네트워크 통신 없이 무료로 이어하기 기능을 제공합니다. 게임 코드에 남는 광고 요청·완료 콘솔 메시지는 실행 중 로컬 진단 메시지이며 외부 서비스로 전송되지 않습니다.

Google Play Android 앱은 다음 원칙으로 동작합니다.

- `android.permission.INTERNET` 권한을 선언하지 않음
- 게임 자산을 `WebViewAssetLoader`의 앱 내부 로컬 오리진에서 제공
- HTTP 평문 통신을 허용하지 않음
- 쿠키, 제3자 쿠키, 위치 기능과 파일·콘텐츠 접근을 허용하지 않음
- 앱 내부 로컬 오리진이 아닌 HTTP·HTTPS 링크는 WebView에서 불러오지 않고 이용자가 선택한 외부 브라우저로 전달

`https://appassets.androidplatform.net`은 외부 웹 서버가 아니라 Android WebView가 앱 패키지 내부 자산을 안전한 방식으로 제공하기 위해 사용하는 로컬 오리진입니다.

앱인토스 버전의 설치·실행 환경과 배포 번들 전달에는 토스 플랫폼의 네트워크와 보안 기능이 사용될 수 있습니다. 이는 게임 코드가 이용자 정보를 운영자 서버로 보내는 것과 구별되며 토스의 정책을 따릅니다.

향후 광고, 분석, 로그인, 결제, 클라우드 저장, 원격 오류 수집 또는 외부 서버 통신을 추가하는 경우 해당 기능을 배포하기 전에 이 방침과 Google Play 데이터 보안 고지를 갱신합니다.

## 8. 배포 플랫폼에서의 별도 처리

Google Play와 토스는 앱·미니앱 다운로드와 실행, 플랫폼 계정, 기기 보안, 서비스 안정성, 부정 사용 방지 또는 법적 의무 이행을 위해 운영자와 별개로 정보를 처리할 수 있습니다. 해당 처리는 각 플랫폼 사업자의 방침을 따릅니다.

- [Google 개인정보처리방침](https://policies.google.com/privacy?hl=ko)
- [토스 개인정보처리방침](https://toss.im/privacy)

Google Play가 제공하는 Android vitals와 Pre-launch report 등 플랫폼 진단 정보는 Google Play 정책과 설정에 따라 처리될 수 있습니다. 앱 자체에는 별도의 원격 오류 수집 SDK가 없습니다.

Google Play의 데이터 보안 섹션은 서비스 등록정보에 표시되는 별도 고지입니다. 운영자는 실제 배포본이 광고·분석 SDK를 포함하지 않고 게임 데이터를 수집·공유하지 않는 현재 구조와 일치하도록 해당 고지를 유지합니다.

## 9. 보유 기간과 삭제 방법

운영자는 이름, 이메일 또는 계정 ID와 연결된 게임 기록을 자체 서버에 저장하지 않습니다. 이용자는 다음 방법으로 기기에 저장된 엔딩 도감과 음소거 설정을 삭제할 수 있습니다.

- Google Play Android 앱: Android 설정에서 `각본대로`의 저장공간·앱 데이터를 삭제하거나 앱 제거
- 앱인토스 버전: 토스 앱에서 해당 미니앱의 저장 데이터를 삭제하거나 토스 앱 제거
- 개발용 브라우저 실행본: 해당 사이트의 Local Storage 또는 사이트 데이터 삭제

삭제한 로컬 게임 데이터는 운영자가 보유한 서버 사본이 없으므로 복구할 수 없습니다.

운영자가 보유한 문의 정보는 제6조의 보유 기간이 끝나면 복구하기 어려운 방법으로 삭제합니다. Google, GitHub, Google Play 또는 토스가 독립적으로 처리하는 정보의 보유와 삭제는 해당 사업자의 개인정보 도구와 방침을 따릅니다.
