## 1. 기본 원칙과 적용 범위

`순간포착!`의 운영 주체인 릴리에스앤씨(공개 개발자명: 릴리게임즈, 이하 “운영자”)는 이용자의 개인정보를 중요하게 생각하며 개인정보 보호법 등 관련 법령을 준수합니다.

이 방침은 다음 배포 환경에 적용됩니다.

- Google Play에서 배포하는 Trusted Web Activity(TWA) Android 앱
- 앱인토스에서 제공하는 게임 미니앱
- https://flash-pose.vercel.app 에서 제공하는 웹 버전

Google Mobile Ads(AdMob) SDK와 User Messaging Platform(UMP)에 관한 조항은 해당 SDK가 실제로 포함되고 광고 기능이 활성화된 Google Play Android 버전에만 적용됩니다. 광고가 없는 이전 Android 버전, 앱인토스 버전과 일반 웹 버전에는 네이티브 AdMob SDK가 로드되지 않습니다.

현재 서비스는 회원가입, 로그인, 인앱 결제, 채팅, 이용자 게시물, 클라우드 저장 또는 운영자 자체 분석 서버를 제공하지 않습니다. 게임의 정답, 오답, 점수와 난이도는 운영자 서버로 전송하지 않습니다.

Google Play, 토스, Google AdMob 등 각 플랫폼 사업자가 자신의 서비스 운영을 위해 독립적으로 처리하는 정보에는 해당 사업자의 개인정보처리방침도 적용됩니다.

## 2. 개인정보 처리 요약

| 구분 | 처리 내용 |
| --- | --- |
| 앱 내 계정·로그인 | 없음 |
| 운영자 자체 서버로 보내는 게임 기록 | 없음 |
| Google Play 광고 | AdMob 전면 광고가 활성화된 Android 버전에서만 제공 |
| 광고 관련 자동 처리 | IP 주소에서 추정되는 대략적 위치, 앱·광고 상호작용, 진단 정보, 기기 또는 기타 식별자 |
| 광고 동의·선택 | UMP를 통해 적용 지역에서 동의를 요청하고 개인정보 선택 수단 제공 |
| 별도 이용 분석 | Google Analytics·Firebase Analytics 사용 안 함 |
| 별도 원격 오류 수집 | Firebase Crashlytics 등 사용 안 함 |
| 위험 권한 | 위치, 연락처, 사진, 카메라, 마이크 권한 요청 안 함 |
| 웹 콘텐츠 제공 | Vercel이 HTTPS 요청 과정에서 IP 주소와 접속 로그를 처리할 수 있음 |

| 이용자가 직접 보내는 문의 | 발신 이메일 주소, 문의 내용과 자발적으로 첨부한 정보 |
| 개인정보 판매 | 운영자가 개인정보를 판매하지 않음 |

비개인 맞춤 광고 또는 제한된 광고도 광고 제공, 빈도 제어, 보안과 부정행위 방지를 위해 IP 주소, 기기 정보 또는 진단 정보 일부를 처리할 수 있습니다. 따라서 “비개인 맞춤 광고”가 모든 데이터 처리가 중단된다는 뜻은 아닙니다.

## 3. Google Play Android 앱의 AdMob 처리

광고가 활성화된 Google Play Android 버전은 게임 10판이 모두 끝난 결과 화면 이후의 자연스러운 화면 전환에서만 Google AdMob 전면 광고를 표시할 수 있습니다. 앱 시작, 포즈 노출, 답 선택, 오답 재도전 또는 게임 진행 중에는 광고를 표시하지 않습니다.

Google의 Google Mobile Ads SDK 공개 자료에 따르면 AdMob SDK는 광고 제공, 광고 성과 측정, 분석, 보안과 부정행위 방지를 위해 다음 정보를 자동으로 수집하고 Google과 공유할 수 있습니다.

| 정보 범주 | 처리될 수 있는 예 | 목적 |
| --- | --- | --- |
| 대략적인 위치 | IP 주소로부터 추정되는 일반적인 위치 | 지역에 맞는 광고·동의 처리, 분석, 부정행위 방지 |
| 앱 활동 | 앱 실행, 탭, 광고 요청·노출·클릭·동영상 조회 등 상호작용 | 광고 제공과 성과 측정, 빈도 제어, 분석 |
| 앱 정보 및 성능 | 앱·SDK 실행 시간, 중단, 성능과 에너지 사용 관련 진단 | 안정성 분석, 오류 대응, 부정행위 방지 |
| 기기 또는 기타 식별자 | Android 광고 ID(AAID), App Set ID와 적용 가능한 계정 관련 식별자 | 광고, 측정, 빈도 제어, 부정행위 방지 |

실제 처리 항목은 이용자의 기기 설정, 지역, 연령 처리 설정, 동의 선택과 Google의 광고 제공 방식에 따라 달라질 수 있습니다. 해당 정보는 전송 중 TLS로 암호화됩니다.

운영자는 M1 광고 버전에서 AdMob 이외의 광고 네트워크, 광고 미디에이션, 보상형 광고, 앱 오픈 광고 또는 배너 광고를 사용하지 않습니다. 별도의 Google Analytics나 Firebase Crashlytics도 사용하지 않습니다.

Google의 처리와 선택 방법은 다음 문서에서 확인할 수 있습니다.

- [Google 개인정보처리방침](https://policies.google.com/privacy?hl=ko)
- [Google 광고 기술에서 사용하는 정보](https://policies.google.com/technologies/ads?hl=ko)
- [Google 서비스에서 수집하는 데이터](https://policies.google.com/technologies/partner-sites?hl=ko)
- [내 광고 센터](https://myadcenter.google.com/)

## 4. 광고 동의와 개인정보 선택

광고가 활성화된 Google Play Android 버전은 Google UMP를 사용하여 적용되는 지역의 동의 또는 개인정보 선택 요건을 확인합니다.

- 앱 실행 시 UMP 동의 정보를 갱신합니다.
- 법령이나 Google 설정상 동의 메시지가 필요하면 광고 요청 전에 메시지를 표시합니다.
- 광고 요청이 허용되지 않은 상태에서는 광고를 요청하거나 표시하지 않습니다.
- 개인정보 옵션이 필요한 상태에서는 앱 안에서 다시 열 수 있는 `개인정보 선택` 진입점을 제공합니다.
- 이용자가 선택을 변경하면 이후 광고 요청에 갱신된 선택을 적용합니다.

이용자는 Android의 개인정보·광고 설정에서 광고 ID를 재설정하거나 삭제할 수 있고, [내 광고 센터](https://myadcenter.google.com/)에서 Google의 맞춤형 광고 설정을 관리할 수 있습니다. 운영자가 직접 보유하지 않는 Google 광고 데이터에 대한 요청은 Google의 개인정보 도구를 이용해야 합니다.

## 5. 웹 콘텐츠 제공과 Vercel 처리

Google Play Android 앱은 TWA 방식으로, 일반 웹 버전은 브라우저로 https://flash-pose.vercel.app 의 게임 콘텐츠를 HTTPS로 불러옵니다. 앱인토스 버전은 앱인토스 배포 번들의 자산을 사용하며 네이티브 AdMob SDK를 로드하지 않습니다.

정적 콘텐츠를 제공하는 Vercel Inc.는 요청 전달, 보안, 장애 대응과 서비스 운영을 위해 다음 정보를 처리할 수 있습니다.

- IP 주소와 IP 주소에서 파생될 수 있는 대략적 위치
- 접속 시각, 요청 URL, HTTP 상태와 전송량
- 브라우저·운영체제·기기 유형을 나타내는 User-Agent 등 요청 헤더
- 서비스 보안과 오류 진단에 필요한 접속 로그

운영자는 웹 페이지에 별도의 추적 픽셀, 행동 분석 도구 또는 맞춤형 광고 쿠키를 넣지 않습니다. Vercel의 처리에 관한 자세한 내용은 [Vercel 개인정보처리방침](https://vercel.com/legal/privacy-policy)에서 확인할 수 있습니다.

## 7. 운영자가 수집하지 않는 정보

운영자는 앱 기능을 통해 다음 정보를 직접 입력받거나 자체 서버에 저장하지 않습니다.

- 이름, 생년월일, 성별과 계정 비밀번호
- 전화번호, 주소와 연락처
- 정확한 위치 정보
- 사진, 동영상, 파일, 카메라와 마이크 입력
- 결제, 금융과 거래 정보
- 건강, 생체정보 등 민감정보
- 게임의 정답, 오답, 점수와 선택 난이도

앱에는 이용자가 이러한 정보를 입력하는 회원가입, 프로필, 채팅 또는 게시물 화면이 없습니다. 다만 이용자가 이메일 문의에 자발적으로 정보를 포함하면 제8조에 따라 처리될 수 있습니다.
